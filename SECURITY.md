# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, please report them by emailing the maintainers or using [GitHub's private vulnerability reporting](https://github.com/ITISFoundation/itis-dakota/security/advisories/new).

## Scope

This project provides Python wheel packaging for [Dakota](https://dakota.sandia.gov/) toolkit. Security issues may fall into:

- **Packaging and build pipeline**: Issues in the CI/CD workflow, wheel building, or distribution process.
- **Upstream Dakota**: Vulnerabilities in the Dakota C++ codebase itself should be reported to [Sandia National Laboratories](https://dakota.sandia.gov/).
