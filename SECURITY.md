# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Instead, please use [GitHub's private vulnerability reporting](https://github.com/ITISFoundation/itis-dakota/security/advisories/new) to submit your report.
3. Include a description of the vulnerability, steps to reproduce, and any potential impact.

We aim to provide a fix or mitigation plan.
